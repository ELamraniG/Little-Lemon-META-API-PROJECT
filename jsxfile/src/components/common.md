# common.js

**What it is for:** Reusable, stateless UI components shared across various pages.
**Technologies used:** React.
**Why:** Promotes code reuse for common elements like loading indicators, alerts, and modals, keeping the main logic files clean and strictly focused on business logic.
**How:** Exports modular functional components such as `Spinner`, `Alert`, `Modal`, and `QtyCtrl` that receive props and render UI conditionally based on those props.
